# Arabman

Translate to arabic numerals to roman numerals.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'arabman'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install arabman

## Usage

```ruby
require 'arabman'
460.to_roman # => 'CDLX'
```
